# 4 Pilares Consultoria - Site Corporativo

Site corporativo profissional para 4 Pilares Consultoria em LGPD, Marco Civil da Internet e ECA Digital.

## Estrutura do Projeto

### ETAPA 1 - Topo do Funil (Awareness)
- ✅ Hero Section com proposta de valor
- ✅ Panorama Regulatório (LGPD, Marco Civil, ECA Digital)
- ✅ Riscos e Consequências da Não Conformidade
- ✅ CTAs para próximas seções

### ETAPA 2 - Meio do Funil (Interest/Desire)
- [ ] Os 4 Pilares da Metodologia
- [ ] Quem Somos (Dr. João Pedro)
- [ ] Ecossistema Completo

### ETAPA 3 - Fundo do Funil (Action)
- [ ] Módulos de Serviços
- [ ] CTAs para WhatsApp
- [ ] Rodapé com Contatos

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Design System

- **Cores Principais**: Azul (#3b82f6), Cinza (#0f172a)
- **Tipografia**: Poppins (headings), Inter (body)
- **Spacing**: Sistema de 20px
- **Breakpoints**: Mobile-first responsive design
