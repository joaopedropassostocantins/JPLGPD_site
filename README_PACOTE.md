# Pacote organizado — documentação do site JPLGPD / 4 Pilares LGPD

Este pacote foi estruturado para facilitar o envio ao repositório GitHub.

## Estrutura sugerida

- `docs/01-visao-geral/` — briefing e visão geral
- `docs/02-site-html/` — artefatos e origem do site
- `docs/03-painel/` — painel, admin e arquitetura visual
- `docs/04-readmes-origem/` — READMEs, TODOs e notas originais
- `docs/05-arquitetura-e-notas/` — sínteses e instruções criadas agora
- `docs/06-checkout-e-pagamento/` — checkout e arquivos de pagamento
- `docs/07-zips-origem/` — preservação dos ZIPs originais

## Passo a passo para subir ao GitHub

1. Baixe e extraia este pacote.
2. Copie a pasta `docs/` para a raiz do repositório `JPLGPD_site`.
3. Revise os arquivos duplicados e mantenha a versão mais atual.
4. Faça commit e push com um token novo.
